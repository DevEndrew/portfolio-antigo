Principais Habilidades

HTML, CSS e JavaScript

Trabalho com a tríade fundamental do desenvolvimento web: HTML para estruturação, CSS para estilo e JavaScript para tornar as páginas interativas e dinâmicas.

Sass

Utilizo o Sass, um pré-processador CSS, para simplificar e modularizar o processo de estilização, tornando o código mais organizado e fácil de manter.

React JS

Utilizo o React JS, uma biblioteca que me permite criar interfaces web modernas e reativas, proporcionando uma experiência de usuário envolvente.

Next JS

Com conhecimentos em Next JS, consigo construir aplicações web escaláveis e otimizadas, facilitando a transição entre páginas e melhorando a visibilidade nos motores de busca.

TypeScript

Trabalho com TypeScript para adicionar tipagem ao JavaScript, tornando o código mais claro, seguro e fácil de entender durante o desenvolvimento.

Node JS

No desenvolvimento full-stack, faço uso do Node JS para criar servidores eficientes, garantindo uma comunicação eficaz entre o frontend e o backend das aplicações web.

MySQL

Manejo bancos de dados com MySQL, garantindo uma organização e recuperação eficientes dos dados necessários para as aplicações.

BootStrap

Implemento o BootStrap para acelerar o desenvolvimento, utilizando sua coleção de componentes prontos para garantir uma experiência consistente em diferentes dispositivos.

Git

No controle de versão, utilizo o Git para rastrear alterações no código, facilitando a colaboração e o gerenciamento de projetos.

Figma

Para o design de interfaces, recorro ao Figma na criação de protótipos visuais, colaborando eficientemente com designers para uma implementação precisa no desenvolvimento final.
